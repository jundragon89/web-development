Instructions on Git

How to host your first site on Github

Checklist:

  1. Ensure that git is installed on your system, and you have a Github account
  2. Open a new public repository with a name of your choice, and copy down the HTTPS url of the git repository
  3. Name the HTML document to index.html (Why index.html?)
  4. Open up the Git CLI and navigate inside the folder where your project file is located
  5. Enter the following commands:

    git init

    git remote add origin (Your link here).git

    git add -A

    git commit -am "Your comment here"

    git push -u origin master

  6. Go to (your github username).github.io/(your repository name)

Note: Git is a version control system and Github is one of the online hosting services using Git. However, how Git actually works and how we can use them are beyond the scope of this workshop. If you'd like to know more about how to use git, you may read up more on this tutorial.
