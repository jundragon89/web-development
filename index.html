Warm Up Session

Let's have some fun with HTML and CSS!

Playing with HTML tags

HTML is the backbone of every website you visit. It defines the structure of a website (for example, paragraphs, images, lists, etc.). It does not define the color, position, or anything related to stylings of a web page.

In this section, you'll create a simple page that describes yourself using HTML tags.

Checklist:

  1. Create heading using the 'h' tag. (Reference)
  2. Include paragraph tags inside the page. (Reference)
  3. Insert link(s) using the anchor tag. (Reference)
  4. Include an image inside a div. We will learn how to position the image later.(Reference)
  5. Create a list (Reference)
  6. Create a table (Reference)
  7. Notice the limitations of HTML. How can you make it more beautiful?
  8. Finished early? Feel free to explore more about HTML, or prepare yourself for the next section by reading up on CSS.

Having fun with CSS

Cascading Style Sheets (CSS) allows you to add styling to your web page. There are a lot of attributes that can be adjusted, like colors, positions, sizes, etc.

Using the previous HTML page you have created, try to style it.

Checklist:

  1. Changing color of text (Reference)
  2. Changing background color/image of a div (Reference)
  3. Adjusting font size, font weight, text decoration (Reference 1) (Reference 2)
  4. Margin and padding (Reference 1: Padding) (Reference 2: Margin)
  5. Positioning the image within a div. Try to experiment with different types of positioning and know the differences. (Reference 1: Float) (Reference 2: Position)
  6. One of the ways to learn how to use CSS is by observing. Right click on any website, and click inspect element. You will be able to view the HTML structure and CSS properties of any given section. Example 1 (Apple) Example 2 (W3School)
  7. Finished early? CSS is a wide topic, as there are infinitely many ways you can style your page. A lot of it comes with experiences, so the more you explore and experiment, the faster you'll learn. Here are some topics you may read up on:
    1. CSS values and units
    2. The box model
    3. Advanced CSS Selectors
